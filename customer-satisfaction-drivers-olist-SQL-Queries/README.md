## SQL Analysis – Customer Satisfaction Drivers

This folder contains SQL queries used to analyze the Brazilian Olist E-commerce dataset and identify key factors influencing customer satisfaction.

### Objective

The goal of this analysis is to explore how delivery performance, pricing, product categories, and customer behavior impact review scores.

### Analyses

1. **Delivery Performance vs Customer Reviews**
   Analyzes how delivery timing (on-time vs late) relates to customer satisfaction.
   → `Brazilian Dataset Question 1.sql`

2. **Order Value and Shipping Cost vs Review Scores**
   Examines how order value and shipping costs vary across different review scores.
   → `Brazilian Dataset Question 2.sql`

3. **Combined Impact of Delivery Timing and Order Value**
   Evaluates how delivery delays and price levels together affect customer satisfaction.
   → `Brazilian Dataset Question 3.sql`

4. **Product Category Performance Analysis**
   Compares revenue, order volume, and average review scores across product categories.
   → `Brazilian dataset Question 4.sql`

5. **Customer Loyalty and Review Behavior**
   Compares review patterns between one-time and repeat customers.
   → `Brazilian Dataset Question 5.sql`

### Notes

* Each SQL file is organized by a specific business question.
* Queries include comments explaining key logic and transformations.
* The analysis focuses on identifying relationships, not causation.
